# Laravel Invoicer

A full-featured Laravel application for managing quotes, invoices, and clients.

## Features

- **Client Management**: Create, update, and manage clients with full contact details
- **Quote Management**: Create quotes, track status (draft, sent, signed), and convert to invoices
- **Invoice Management**: Create invoices, track payments, send via email
- **PDF Generation**: Professional PDF documents for quotes and invoices
- **Email Integration**: Send quotes and invoices directly via email
- **Brand Customization**: Company logo, colors, default terms, and more
- **Dashboard**: Overview with charts and recent activity
- **Responsive Design**: Works on desktop and mobile

## Requirements

- PHP 8.2+
- MySQL 8.0+
- Node.js 18+
- Composer
